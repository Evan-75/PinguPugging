# PinguPugging – Det periodiske system

Offline‑first quiz- og puggeapp for det periodiske systemet.

## Funksjoner
- Quiz og filtrering av grunnstoffer
- Mobil- og desktoptilpasset UI
- Installerbar som PWA
- Full offline‑støtte etter første innlasting

## Bruk
Appen er ment å brukes via HTTPS (f.eks. GitHub Pages).
Åpne appen én gang online → deretter fungerer den offline.

## Lokal utvikling
```bash
python -m http.server 8000
```
Merk: Service workers fungerer ikke via lokal IP over HTTP.
